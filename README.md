##Jogo do Número Secreto

Este projeto consiste em um jogo interativo de adivinhação desenvolvido em **JavaScript**, no qual o usuário deve descobrir um número secreto gerado aleatoriamente dentro de um intervalo definido.
__________________________________________________________
##Sobre o projeto

O objetivo do jogo é desafiar o usuário a encontrar o número correto com base em tentativas e dicas fornecidas pelo sistema. A cada tentativa, o jogo informa se o número secreto é maior ou menor, até que o jogador acerte.

Durante o desenvolvimento, foram aplicados conceitos fundamentais de programação, incluindo:

* Manipulação do **DOM**
* Uso de **funções**
* **Estruturas condicionais**
* Lógica com **arrays**
* Geração de **números aleatórios**
* Controle de repetição de números sorteados

Além disso, foi implementada uma funcionalidade que evita a repetição de números até que todos os valores possíveis sejam utilizados, tornando o jogo mais dinâmico.
_____________________________________________________
##Acessibilidade

O projeto utiliza a **Web Speech API (ResponsiveVoice)** para reprodução de mensagens em áudio, proporcionando uma experiência mais interativa e acessível ao usuário.

Tecnologias utilizadas
* HTML
* CSS
* JavaScript

 Obs: Este projeto foi desenvolvido como parte de um curso, com foco na implementação da lógica utilizando JavaScript.
